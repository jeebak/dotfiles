# ESLint

_"The pluggable linting utility for JavaScript and JSX"_ 

```
npm install -g eslint
eslint --init

# Recursively linting a directory
eslint "foo/**/*.js"

# Autofix Errors
eslint "foo/*.js" --fix
```
